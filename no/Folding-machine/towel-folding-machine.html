Håndklebrettemaskin

  • produkt detaljer
  • Spesifikasjoner


1.Spesiell design for ulike vaskemaskiner: Denne maskinen er vår spesialdesign fra fabrikken - et profesjonelt automatiseringsutstyr for håndklebretting. Mappen kan levere mange forskjellige typer foldemåter, som tar sikte på å øke brettehastigheten og øke arbeidseffektiviteten.
2. Maks foldehastighet: kan nå 50 m/min. 1800 stykker stort håndkle/time. kan øke arbeidseffektiviteten 3 ganger.
3.Various Folding Ways (width folding+cross folfing) : (1/3+0),(1/3+1/3),(1/3+1/2),(1/3+1/4),(1/2+0),(1/2+1/2),(1/2+1/3),(1/2+1/4),
(1/1+0),(1/1+1/2),(1/1+1/3),(1/1+1/4).
4. Struktur for ulike vaskemaskiner: Vedta avansert idé for å designe strukturen, fast og stødig.
5.Importerte deler: De viktige reservedelene og pneumatiske komponentene bruker det internasjonale toppmerket.som NSK-lager.som gjør maskinen pålitelig.
6.Perfekt menneske-maskin-grensesnitt: Enkel å betjene. Alle typer arbeidsinformasjon, inkludert størrelsene på lin, foldeformen, plasseringen av lin kan reageres på brukeren, for daglig drift og vedlikehold.
7. Overføringssystem: Hastighet lukket sløyfefunksjon, elektrooptisk hastighetsmålingssystem,
overføringssystem essensielle komponenter importeres. 8. Sterk struktur: Garanterer påliteligheten og høy effektivitet 1,800 stykker håndkle / time.
9.High Intelligence PLC Touch Sreen: Enkel å kontrollere funksjonsfeilen og enkel å betjene. 10. Folding standard distinksjonsfunksjon: Beregner bretteomfanget, nekter å brette det brede håndkleet, beskyttelse og alarm, garanterer brettekvaliteten.
